# investments-vault

Private archive for Market Logic Investors LLC. Working files for live deals, theses, diligence, and frameworks.

## Structure

- `vc-deals/` — active and archived VC memos, one folder per company
  - `innerwell/` — Series A digital mental health (active IC)
- `pe-deals/` — PE buyout, growth, secondaries memos
- `industry-theses/` — long-form sector views with watchlists
- `frameworks/` — master copies of scorecards, checklists, and templates

## Companion public site

Anonymized or methodology-only versions of select work are promoted to the public site:

- Repo: `marketlogicinvestors/marketlogicinvestors.github.io`
- URL: https://marketlogicinvestors.github.io

## Confidentiality policy

- Everything in this repo is private and may contain NDA or deck-derived information.
- Do not copy, paste, summarize, or share externally without explicit redaction.
- Only Market Logic Investors LLC members and named collaborators have access.
- Founder correspondence, pitch decks, and PitchBook exports stay inside their deal folders.

## Workflow

1. Create a folder under the right top-level directory.
2. Add `memo.md` plus supporting files using the per-deal layout in each subfolder README.
3. Use frameworks from `frameworks/` rather than writing from scratch.
4. When a piece is ready for public sharing, create a redacted copy and PR it to the public site repo. Never push raw deal content to the public site.

## Access

Managed via the `marketlogicinvestors` GitHub organization. Add collaborators per repo, not per file.
