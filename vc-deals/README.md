# VC Deals

Confidential venture deal memos and supporting files. Organize one folder per company.

## Suggested layout per deal

- `memo.md` or `memo.html` — IC memo
- `deck/` — original pitch deck
- `diligence/` — DD questions, responses, references
- `data/` — financial model, cohort data, payer mix
- `correspondence/` — founder and reference emails
- `notes.md` — call notes and follow-ups

## Active deals

- innerwell/ — Series A digital mental health (active IC)

## Confidentiality

All content is private. Do not copy, paste, or summarize externally without redaction.
